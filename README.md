# GroundFloor Prototype

Static prototype for testing a fan-powered music discovery platform where supporters nominate, follow, and help emerging artists build visibility.

## Prototype Flow

- Landing page
- Discovery / trending artists
- Fan feed
- Artist campaign page
- Create or nominate an artist page
- Supporter profile
- Auth mockup

## Phase 1 Goal

Use this prototype to gather early feedback from artists, fans, and potential supporters before investing in a full product build.
